## Story

**As a** user  
**I want to** see helpful messages when there's no content  
**So that** I understand why and what I can do

---

## Acceptance Criteria

1. Empty state UI is displayed when no results found
2. Message explains why content is empty
3. Suggestions provided for what to do next
4. Visually consistent with site design
5. Works for: empty search, empty category, no components

### Component Architecture

**EmptyState Component:**

```typescript
interface EmptyStateProps {
  icon?: React.ComponentType;
  title: string;
  message: string;
  actions?: Array<{
    label: string;
    onClick: () => void;
    variant?: "primary" | "secondary";
  }>;
}
```

**Example Usage:**

```typescript
<EmptyState
  icon={SearchIcon}
  title="No results found"
  message="Try adjusting your search or filters"
  actions={[
    { label: "Clear filters", onClick: handleClearFilters },
    { label: "View all", onClick: handleViewAll }
  ]}
/>
```
