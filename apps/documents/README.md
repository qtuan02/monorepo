# @monorepo/documents

Vite + React documentation site for UI components and hooks in this monorepo.

**Full guide:** [docs/apps/DOCUMENTS.md](../../docs/apps/DOCUMENTS.md)

## Quick start

```bash
# From monorepo root
pnpm dev:documents
```

Dev server: **http://localhost:3000** (see `vite.config.ts`).

## Environment

Optional `VITE_*` keys live in the **repo root** `.env`. Use `import { env } from "@monorepo/env/vite"` in code — see [`@monorepo/env`](../../packages/env/README.md).
